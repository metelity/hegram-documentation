Overview

Hegram is a powerful Python library for building Telegram bots with a clean and intuitive API.

Installation

Cloning repositories

git clone github.com/metelity/hegram

Choice of the directory

cd hegram

Started-installing

python install.py

Getting Started

A Simple Bot Example

from hegram import Bot

bot = Bot(token="YOUR_TOKEN")

@bot.function("/start")
def start_handler(update):
    return "Hello, world!"

bot.run()

API Reference

The Bot Class

Initialization

Bot(token: str, polling_interval: float = 1.0)

Message Sending Methods

Method Description
send_message() Sends a text message
edit_message() Edits an existing message
delete_message() Delete a message

Media Methods

Method Description
send_photo() Sends a photo (file or URL)

Examples

Inline Keyboard Example

from hegram import Bot

bot = Bot(token="YOUR_TOKEN")

@bot.function("/start")
def start_handler(update):
    chat_id = update["message"]["chat"]["id"]
    keyboard = {
        "inline_keyboard": [
            [{"text": "Option 1", "callback_data": "opt1"}]
        ]
    }
    bot.send_message(chat_id, "Choose an option:", reply_markup=keyboard)

bot.run()
