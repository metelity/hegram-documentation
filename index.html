Overview

Wetchgram is a powerful Python library for building Telegram bots with a clean and intuitive API.

Installation

Cloning repositories

git clone github.com/metelity/wetchgram

Choice of the directory

cd wetchgram

Started-installing

python install.py

Project Structure

Getting Started

A Simple Bot Example

from wetchgram import Bot

bot = Bot(token="YOUR_TOKEN")

@bot.function("/start")
def start_handler(update):
    return "Hello, world!"

bot.run()

API Reference

The Bot Class

Initialization

Bot(token: str, polling_interval: float = 1.0)

Message Sending Methods

MethodDescription
send_message()Sends a text message
edit_message()Edits an existing message
delete_message()Delete a message

Media Methods

MethodDescription
send_photo()Sends a photo (file or URL)
edit_message_media()Edits message media
edit_message_caption()Edits message caption
copy_message()Copies message to another chat

Chat Management

MethodDescription
get_chat()Get chat information
get_chat_administrators()Get chat admins list
export_chat_invite_link()Generate invite link
set_chat_title()Change chat title
set_chat_photo()Change chat photo

Webhook Methods

MethodDescription
set_webhook()Set webhook URL
delete_webhook()Remove webhook
get_webhook_info()Get webhook status

Games Methods

MethodDescription
send_game()Send a game
set_game_score()Set player score
get_game_high_scores()Get high scores

Stickers & Dice

MethodDescription
send_sticker()Send sticker
send_dice()Send dice/emoji

Forum Topics

MethodDescription
create_forum_topic()Create new topic
reopen_forum_topic()Reopen closed topic

Examples

Webhook Setup Example

bot.set_webhook(
    url="https://yourdomain.com/webhook",
    max_connections=40
)

Chat Management Example

# Get chat info
chat_info = bot.get_chat(chat_id)

# Change chat title
bot.set_chat_title(chat_id, "New Chat Title")

# Export invite link
invite_link = bot.export_chat_invite_link(chat_id)

Game Example

# Send game
bot.send_game(
    chat_id=chat_id,
    game_short_name="my_game"
)

# Set player score
bot.set_game_score(
    user_id=user_id,
    score=100,
    chat_id=chat_id,
    message_id=message_id
)
